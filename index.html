About Blocksmith

Blocksmith.tools is a free, browser based toolkit for Minecraft builders. No downloads, no accounts, no ads that follow you around. Just tools that help you build.

We offer pixel perfect generators for circles, ovals, spheres and arcs, with block by block progress tracking, real Minecraft block textures, and direct export to .schem schematics compatible with WorldEdit, Litematica and Axiom.

This project started when one Minecraft player got tired of outdated, clunky generator sites and decided to build something better. It has grown into a full toolkit. We're always adding new tools. If you have a suggestion, let us know!

How to Build a Perfect Circle in Minecraft

Building a perfect circle in Minecraft is one of the most common challenges for builders. Because Minecraft uses a block based grid, true curves don't exist. Instead, you approximate them using a technique called pixel art circles or voxel circles.

The trick is to follow a mathematical pattern where each row of blocks is offset slightly from the previous one. The Blocksmith Circle Generator calculates this pattern automatically for any diameter you choose, from tiny 5-block circles to massive 200-block rings.

Tips for building circles:

  • Use odd diameters for perfectly symmetrical circles with a clear center block
  • The "Outline" mode shows just the ring, ideal for towers, arenas and portals
  • The "Filled" mode fills the entire circle, great for platforms and floors
  • Click any block to mark it as placed and track your progress
  • Export as .schem to paste directly into your world with WorldEdit or Litematica

How to Build a Sphere in Minecraft

A Minecraft sphere is built layer by layer. Each horizontal slice of the sphere is a circle with a different radius, largest at the equator and smallest at the top and bottom. The Blocksmith Sphere Generator shows you exactly which blocks to place on each layer.

Use the layer slider to move through each cross-section of the sphere from bottom to top. The 3D viewer lets you rotate, pan and zoom to visualise the full shape before you start building.

Tips for building spheres:

  • Start from the bottom layer and work upward
  • Use the "Below" view in the 3D viewer to see all completed layers
  • Hollow spheres are great for domes, planetarium builds and giant sculptures
  • For large spheres (50+ diameter), export as .schem and use a paste tool. Building manually takes hours
  • Combine multiple sphere layers to create eggs, blimps or organic shapes

Oval and Arc Generators

Not every build needs a perfect circle. The Oval Generator lets you set independent width and height values to create stretched or compressed ellipses, perfect for race tracks, swimming pools, stadium designs and natural-looking terrain.

The Arc Generator creates partial circles and curved walls. Set the start and end angle to get exactly the curve you need. Arcs are ideal for curved bridges, amphitheatres, curved roads and decorative architectural details.

Common uses:

  • Oval: Swimming pools, race ovals, oval windows, natural cave entrances
  • Arc: Bridge arches, curved walls, half-pipe ramps, decorative cornices
  • Both tools support block selection and .schem export
  • Use the zoom view to inspect fine details at high diameter values

Pixel Art in Minecraft

The Pixel Art tool lets you upload any image and convert it into a Minecraft block layout automatically. The tool analyses each pixel and finds the closest matching Minecraft block colour from a palette of 162 blocks including concrete, wool, terracotta, glazed terracotta, and more.

You can also paint pixel art manually using the brush and fill tools. Set any custom width and height up to 256×256 blocks, choose your palette, and export the finished design as a .schem file ready to paste into your world.

Tips for pixel art:

  • Use concrete blocks for the most vibrant and consistent colours
  • Keep pixel art below 128×128 for practical in-game building
  • The fill tool (paint bucket) is great for large background areas
  • Wool gives a slightly different shade than concrete. Mix them for shading effects

Frequently Asked Questions

What is a .schem file and how do I use it?

A .schem file (schematic) is a file format used by Minecraft building tools like WorldEdit, Litematica, and Axiom. Download the schematic from Blocksmith Tools, place it in your tool's schematics folder, and paste it into your world at any position. This saves hours of manual block by block building.

Does this work for Minecraft Java and Bedrock?

The circle, oval, sphere and arc generators work identically for both Java and Bedrock editions. The block patterns are the same. The .schem export format is designed for Java Edition tools. Bedrock players can still use the visual guides to build manually.

What diameter should I use for a sphere?

For decorative builds, diameters between 15 and 30 blocks look great and are manageable to build. Go bigger and the results get even better. Just export as a .schem and paste it in with WorldEdit, Litematica or Axiom. There is no limit on what you can build.

Why don't Minecraft circles look perfectly round?

Minecraft uses a cubic block grid, so true curves are impossible. What we call a "Minecraft circle" is the closest possible approximation using whole blocks, similar to how old pixel art represents curves on a screen. The generator uses the midpoint circle algorithm to produce the most accurate approximation possible at any given diameter.

Is Blocksmith Tools free?

Yes, completely free, no account required, no downloads. All tools run directly in your browser. The only optional feature is the .schem export which also costs nothing.

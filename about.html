About Blocksmith Tools

It started one time I was playing Minecraft a while ago. I was trying to make a big circle for a castle I had in mind. When I looked up tools online I found that every single one of them had issues, glitches, old interfaces.

So I decided to make my own tool. At some point I mentioned it to my friends when we were playing on a server together and they actually started using it. So I said to myself "Why not make it a website? Maybe it can actually help more players." And that's how we got here. :)

Enjoy the site and I hope it serves you well!

What Blocksmith Tools Is

Blocksmith Tools is a free, browser based collection of building aids for Minecraft. No account, no download, no subscription. Just open it and start building smarter.

Every tool is built around one idea: builders should spend their time building, not doing maths.

⭕

Circle Generator

Perfect circles at any diameter. Outline or filled. Click to track progress.

🥚

Oval Generator

Independent width and height for ellipses, pools, race ovals and organic shapes.

🌐

Sphere Generator

Layer by layer sphere guide with a 3D viewer. Hollow or solid.

🌙

Arc Generator

Curved walls, bridge arches and custom angle arcs at any diameter.

🎨

Pixel Art

Upload any image and convert it to Minecraft blocks automatically. 162 block palette.

⬇

.schem Export

Export any design as a schematic file compatible with WorldEdit, Litematica and Axiom.
